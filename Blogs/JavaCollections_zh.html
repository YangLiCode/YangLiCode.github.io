Collections和Map

Java集合有2个最基础的接口: Collections和Map. Collections中都是单个元素的list/set. Map都是键值对.

我们经常这么写代码:

List myList = new ArrayList<>();

上面这行代码创建了一个新的ArrayList对象，并把它复制给List变量. 我们之所以这么写, 是因为如果我们以后想用其他的List实现类，我们可以方便的改代码.

因为Collections是所有集合的接口, 它的所有方法都能在子类中调用. 下面是一些常有的方法:

methods Descriptions
public boolean add(E e) 在集合中添加一个元素
public void clear() 删除集合中的所有元素
public boolean remove(E e) 删除集合中的一个元素
public boolean contains(Object obj) 如果集合中有这个元素，就返回true
public boolean isEmpty() 如果集合中是空的，就返回true
public int size() 返回集合中元素的个数
public Object[] toArray() 以数组的形式返回集合的所有元素
